SugarCream by fm100

Scrum based software project management tool.

SugarCream is a software project management tool. It is totally based on scrum process, and is only for scrum. SugarCream supports backlog, sprint concepts, and some convenient features like daily scrum.

Dependencies

python, django, bootstrap, jquery
